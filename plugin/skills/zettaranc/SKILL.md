# Zettaranc 交易体系 Skill

本技能提供 Zettaranc（知行小菜鸟）A股交易体系的完整知识，可用于查询交易规则、分析买卖信号、摄入新素材到知识库。

## 体系总览

Zettaranc 交易体系由 Z 哥（知行小菜鸟）创建，是一套**规则驱动、纪律优先**的 A 股短线/波段交易系统。核心理念：

- **守株待兔**：只在固定买点（B1/B2/B3）进场，不追高
- **止损铁律**：买入 K 线最低点向下 3-5 个价位，幅度 ≤3%
- **仓位纪律**：超市模式分散持仓，单票不超总仓位 20%

### 数据源检测（按优先级）

1. **环境变量**：`ZETTARANC_KB_PATH` 存在 → 指向本地知识库根目录
2. **本地同级目录**：plugin 同级目录存在 `wiki/` → 本地知识库模式
3. **远程访问**：都没有 → 通过 GitHub API 访问仓库 `chenleiitaz/zettaranc-knowledge`

检测方法：
```bash
# 1. 检查环境变量
echo $ZETTARANC_KB_PATH

# 2. 检查本地同级目录（相对 plugin 安装位置）
ls wiki/zettaranc/index.md 2>/dev/null && echo "本地知识库可用"

# 3. 检查 GitHub 远程仓库
gh repo view chenleiitaz/zettaranc-knowledge --json name 2>/dev/null && echo "远程仓库可用"
```

### GitHub 远程访问方式

当本地知识库不可用时，通过 `gh api` 读取远程文件：


> 如果文件名不确定，先读 `index.md` 获取完整列表。

## 查询工作流

当用户提出交易相关问题时，按以下流程回答：

1. **判断问题类型**：属于哪个分类（信号/战法/工具/心法/筹码）
2. **加载对应 Reference**：按需读取 `references/` 下的对应文件
3. **综合回答**：基于 reference 内容给出精准回答，标注信息来源
4. **交叉引用**：主动关联相关概念（如问 B1 时关联 B2、止损、仓位）

### 回答规范

- 使用**简体中文**
- 关键规则用 **加粗** 标注
- 涉及数值阈值时给出精确数字（如 J<13、涨幅30%、止损3%）
- 如知识库无相关内容，明确声明"知识库中无此内容"，再给出一般性建议
- 回答超过 2 段且有分析价值时，主动询问是否保存为知识页

## 关键规则速查

### 买入信号层级

| 信号 | 核心条件 | 周期 | 优先级 |
|------|---------|------|--------|
| **B1 建仓波** | KDJ J值 < 13 | 日线/周线 | 最高（守株待兔） |
| **B2 突破** | 确认阳线，趋势确立 | 日线 | 高 |
| **B3 买点** | 中继确认，加仓信号 | 日线 | 中 |

### B1 核心铁律

- J值 < 13（数值越低越好，-10、-14 都算）
- **两个30%原则**：涨幅约 30%，累计换手率 ≤30%
- 换手率 >40% = 废 B1
- **3天纪律**：买入后 3 天内必须恢复上涨，否则减仓
- **止损**：买入 K 线最低点向下 3-5 个价位，幅度 ≤3%

### 卖出信号

- **S1 信号**：高位放量滞涨，主力出货迹象
- **DSZ 战法**：顶背离 + 缩量 + 高位十字星组合

### 仓位管理

- **超市模式**：分散持仓，单票 ≤20%
- **黄金坑公式**：(颈线-坑底)+颈线 = 目标价
- **不追高**：只在 B1/B2/B3 进场，其他位置无依据
