# Obsidian 设置指引

## 1. 打开知识库
在 Obsidian 中点击 "Open folder as vault" → 选择 `D:\LLM-Wiki-Vault\`

## 2. 附件路径设置
Obsidian 设置 → 文件与链接：
- **附件默认存放路径** → 设置为 `assets/`（固定目录）
- **附件链接格式** → 使用 `![[文件名称.png]]` 格式

## 3. 下载图片快捷键
- 设置 → 快捷键 → 搜索"下载" → 找到"下载当前文件的附件"
- 绑定快捷键（如 `Ctrl+Shift+D`）
- 剪藏文章后按快捷键，所有图片下载到本地磁盘

## 4. 安装 Obsidian Web Clipper
- 浏览器扩展：https://obsidian.md/clipper
- 设置剪藏笔记保存路径为 `D:\LLM-Wiki-Vault\raw`（或 raw 下的子目录）
- 支持 Chrome、Edge、Firefox 等浏览器

## 5. 推荐插件
- **Web Clipper**：网页剪藏（必需）
- **Marp**：Markdown 幻灯片生成
- **Dataview**：基于 YAML frontmatter 的查询和动态表格
- **Canvas**：思维导图和概念图
- **Excalidraw**：手绘风格图表

## 6. 关系图谱
Obsidian 自带的"关系图谱"视图是查看 Wiki 形态的最佳方式——可以看到哪些页面是核心枢纽、哪些是孤立页面。
