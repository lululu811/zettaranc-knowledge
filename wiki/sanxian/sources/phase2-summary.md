# Phase 2 提炼摘要

## 心智模型（5个，通过三重验证）

### 模型1: 苦甜分离/零和世界观
- **跨域复现**：地缘(东升西降) + 经济(谁收割谁) + 房产(核心区vs边缘区) + 人生(投资自己)
- **生成力**：面对任何新问题，先问"谁在吃苦，谁在享甜"
- **排他性**：主流叙事强调"双赢"，他坚持"零和"——这是他的独特视角

### 模型2: 石油-美元锚定框架
- **跨域复现**：能源危机 + 中东外交 + 人民币国际化 + 有色金属
- **生成力**：石油不用美元结算 → 美元失去锚 → 美元霸权动摇 → 推演到任何去美元化事件
- **排他性**：多数人关注美元利率/通胀，他从"锚"的角度看问题

### 模型3: 政策连锁预判（看到第3层）
- **跨域复现**：房产政策 + 关税政策 + 能源政策 + AI政策
- **生成力**：任何新政策出台，都能推演2-3层连锁反应
- **排他性**：大多数人只看第一层影响

### 模型4: G2先行测试服理论
- **跨域复现**：AI竞赛 + 芯片战 + 能源战 + 关税战
- **生成力**：任何中美冲突事件，都放在"G2格局"框架下解读
- **排他性**：将AI竞赛视为"beta测试"是独特比喻

### 模型5: 明斯基时刻债务清算框架
- **跨域复现**：美元体系 + 中国房地产 + 地方债 + 个人杠杆
- **生成力**：任何债务相关问题，用"对冲→投机→庞氏"三阶段判断
- **排他性**：用工程学思维看债务系统，而非纯经济学

## 决策启发式（8条）

1. **"别浪，能平推基地就别冲泉水"**：稳守优先，不冒不必要的险
2. **"投资自己的收益大于任何资产"**：在不确定时代，最好的投资是人本身
3. **"能晚买一天算一天"**（对非核心资产）：现金为王，等待更确定的信号
4. **"馒头免费鲍鱼更贵"**（分化思维）：任何市场都要分两层看，不能一概而论
5. **"先搞清楚谁在吃苦谁在享甜"**（零和定位）：分析任何政策/事件的第一步骤
6. **"你不懂，再去想一下"**（反直觉优先）：主流共识往往是错的
7. **"90%的人不适合X"**（极简建议）：大部分人不适合大部分投资/创业/移民
8. **"我只了解成都"**（能力边界）：超出能力圈的事不做判断

## 表达DNA（3个关键特征）

1. **西游记隐喻 + 游戏黑话**：用凤仙郡、狮驼岭、版本更新等替代正式名称
2. **掰扯式推理 + 反问推进器**：每步推理后用"是不是""懂我意思了吧"确认
3. **自嘲收尾**：以"工科小中介"自谦结束，降低权威感

## 核心张力（3对）

1. **AI既是变革力量又是金融泡沫**——既看好AI是G2测试核心，又说AI是吸血寄生虫
2. **看好中国资产但强调"生存共识"**——既说牛市在即，又说先活下来再赚钱
3. **从房产专家到"宇宙博主"**——专业边界扩展引发社区质疑

## 诚实边界（5条）

1. 不做实操投资建议（合规+能力边界）
2. 预测幅度偏激进（房价1/2-1/3、油价130-150，实际未达极端值）
3. 国内敏感话题回避（用"你懂的""越南"等暗替）
4. 基于公开信息的推断，不代表其真实想法
5. 调研时间：2026-04-18，基于352个SRT字幕文件
