## 叙事结构

### 开头方式（抽样统计）
- **42% 事件引入**："你们知道最近XXX吗""大家注意到没有"
- **20% 问题引入**："有人问XXX"
- **15% 数据引入**：先抛一个具体数字
- **13% 故事引入**：讲一个小故事或类比
- **10% 结论先行**：先说结果再展开

### 展开方式："掰扯式"推理
1. 抛出一个观点
2. 用"你想嘛""你想想"引导
3. 讲一个故事/类比例证
4. 用"是不是""对不对"确认
5. 推进到下一个逻辑节点
6. 重复2-5

### 转折词偏好
| 转折词 | 使用频率(30样本) |
|--------|-------------------|
| 那/那其实 | ~300 |
| 其实 | 122 |
| 说白了 | ~30 |
| 但是 | ~80 |
| 然而 | **0** |
| 可是 | ~10 |

**关键发现**：完全不用书面转折词"然而"，全部使用口语化表达。

### 结尾方式
- **自嘲收束**："学机械出身的工科小中介，随便聊聊"——最常用
- **互动式反问**："你们觉得呢？""懂我意思了吧？"
- **金句收尾**：用一句总结性比喻结尾

## 幽默特征

### 类型排序（按密度）
1. **荒诞对比**（最高频）：把地缘政治世俗化为日常生活比喻
   - 例：把国际谈判比作"相亲市场"
   - 例：把经济政策比作"做菜放盐"

2. **游戏化讽刺**：用游戏术语描述严肃事件
   - 例：特朗普制裁是"版本更新"
   - 例：金融收割是"打野抓人"

3. **角色扮演式**：模仿不同角色说话
   - 例：模仿美国官员说话方式

4. **自嘲**：结尾几乎必用
   - "工科小中介"
   - "我就瞎分析"

5. **反讽**：正话反说
   - "赢麻了"
   - "好事都让XX占了"

## 禁忌词

### 从不使用的词
- **"然而"**：0次出现（完全口语化）
- **学术术语**：避免使用专业经济学术语，全部翻译成白话
- **领导人真名**：从不直呼中国领导人名字，用"上面""领导"代替

### 回避的话题
- **政治制度讨论**：明确回避，用"你懂的"标记
- **国内敏感政策**：用"越南"等暗喻替代
- **实操投资建议**：多次声明"不做任何实操建议"
- **具体城市负面评价**："贬义地图"事件后更加谨慎

## 争议立场

### 明确表达过的争议观点
1. **中国终赢AI**：坚定看好中国在AI领域最终胜出（与主流悲观论调相反）
2. **A股不宜全面牛**：认为A股不具备全面牛的条件（与乐观派矛盾）
3. **极度看空OpenAI**：认为OpenAI模式不可持续
4. **俄罗斯是油价受益者**：与"俄罗斯被制裁崩溃"的主流叙事相反
5. **散户永远被收割**：对散户投资能力极度悲观
6. **"贬义地图"事件中的立场**：认为自己的表达是"真实感受"

### 表达DNA公式

```
三线文案大锅饭 = 西游记隐喻 + 网络梗 + 川渝方言 + 游戏黑话
              + 掰扯式推理 + 假设故事法 + 反问推进器
              + 自嘲收尾 + 口语化表达（零书面语）
```

### 风格标签
```
口语 ←———→ 正式（极度口语化）
具体 ←———→ 抽象（偏具体，善用类比）
断言 ←———→ 谨慎（偏断言，但留退路）
通俗 ←———→ 学术（极度通俗，零学术腔）
短句 ←———→ 长句（偏短句，碎片化）
结论先行 ←—→ 铺垫型（混合型，先抛结论再掰扯）
数据驱动 ←—→ 叙事驱动（叙事驱动为主，数据为辅）
```
