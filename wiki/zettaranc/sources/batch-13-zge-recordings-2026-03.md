### 3. 260318 砖型图的核心用法(周三晚八)

- **核心主题**: [[砖形图]] 工具的"母图化"精讲 — 把砖型图从一个 K 线辅助升格为"短线/超短线交易主仪表盘",首次系统化提出"四块砖循环""下影线锤子不做""4 天是一个转折循环点"。
- **关键概念**: [[砖形图]] / [[四块砖交易体系]] / [[B1建仓波]] / [[B2突破]] / [[关键K]] / [[转折点战法]]
- **新概念候选**(★ 高优先级):
  - **"四天转折循环点"** — 农历周期 / 潮汐 / 4 年大循环 / 4 天小循环的一种"自然节律"叠加论,Z 哥用大量时间论证"四天是 A 股最朴素的最小循环单元"
  - **"下影线锤子不做"原则(精细化版)** — 区分"锤子"(下影线 + 实体小)、"下影线十字星"(下影线 + 十字星)、"长下影 K 线"(单纯长下影);其中"锤子是 B2 不做,B1 锤子可能是收回筹码"是一个细化分类,可补到 [[砖形图]] / [[关键K]] 页
  - **"四块砖之后的应对"** — 数到第四块砖往后的减仓 / 持有 / 加码三种动作树,与 [[四块砖交易体系]] 互补
- **核心观点**:
  - [[砖形图]] 不是辅助,是 Z 哥短线/超短线交易的"主仪表盘"
  - 一律数四块砖,数完之后再讨论方向(简化决策)
  - 锤子(尤其是 B2 锤子)代表外部有未知筹码砸出,坚决不做

### 4. 260322 砍柴,挑水,做饭(周日晚八)

- **核心主题**: "投资做的是交换信仰" + 下礼拜应对策略;整场围绕一个核心动作 — **"寻找低位筹码换低位筹码的机会"**(回抽黄线动作)。
- **关键概念**: [[筹码战争]] / [[筹码三段论]] / [[去弱留强]] / [[白线黄线系统]] / [[四块砖交易体系]]
- **新概念候选**(★ 高优先级):
  - **"低位筹码换低位筹码"原则** — 整场最高频金句(出现 100+ 次):**不要在相对低位丢掉筹码,要在低位之间做调仓换股,绝不允许低位筹码混到高位筹码里**;是 [[去弱留强]] 的执行细则
  - **"砍柴挑水做饭"操作隐喻** — 把交易类比为"砍柴(选股)、挑水(择时)、做饭(应对)"三个朴素动作,标题级金句
  - **"投资做的是交换信仰 / 交换认知"** — Z 哥首次明确提出市场博弈的本质是"信仰交换"而非"价值发现",可补到 [[A股博弈本质]] 页
- **核心观点**:
  - 调仓的硬约束:**低位换低位**,而不是低位换高位
  - 浮亏不卖、低位筹码绝对保全,是穿越熊牛交替的核心生存原则
  - 砍柴/挑水/做饭隐喻强调"基础动作日复一日,不靠英雄主义"

### 5. 260325 砖型图三种定式(周三晚七点半)

- **核心主题**: 在 0318 砖型图核心用法基础上,**首次系统化提出"砖型图三种定式"** — 这是本批次最高密度的新增工具内容。
- **关键概念**: [[砖形图]] / [[四块砖交易体系]] / [[N型结构]] / [[B1建仓波]] / [[B2突破]] / [[去弱留强]] / [[转折点战法]]
- **新概念候选**(★★ 最高优先级):
  - **砖型图三种定式** — 一个总名称概念页:
    - **定式一:N 型起跳** — 以 [[N型结构]] 为底层,在 N 型尾部做四块砖
    - **定式二:标准四块砖** — 经典连续四根红砖结构
    - **定式三:转折定式** — 跌势中四块砖反转的特殊结构
  - **"高位低位是动态相对概念"细化** — Z 哥强调"没有绝对高位低位,只有相对高位低位",是 [[去弱留强]] / [[筹码三段论]] 的执行细化
  - **"基于四块砖的去弱留强"** — 把四块砖与去弱留强首次合并:数四块砖之后必须做去弱留强动作树
- **核心观点**:
  - 三种定式是 [[砖形图]] 工具的"考试公式" — 不需要再 case-by-case 分析,直接套定式
  - 定式一最常见,定式三最稀缺但赚最多
  - 即使前面不完美,只要后面四块砖标准,定式一样成立

### 6. 260329 全面推演美军若地面战取胜概率(周日晚八)

- **核心主题**: 标题是地缘政治推演,但**实际直播 75% 篇幅讲交易** — 大盘主力完全控盘 / 主仓出货 / 中报概率分析 / 25% 概率窗口的逻辑;后 25% 才是"美军本土地面战"概率推演(地缘政治娱乐性内容,与交易体系关联弱)。
- **关键概念**: [[活跃市值]] / [[筹码战争]] / [[主力出货五种经典方式]] / [[B1建仓波]] / [[B2突破]] / [[A股博弈本质]]
- **新概念候选**:
  - **"主力完全控盘"信号** — Z 哥指出 2026-03 末的大盘已经进入主力完全控盘阶段(可补到 [[A股博弈本质]] / [[AI控盘指数论]])
  - **"中继跳空 / 衰竭跳空 / 攻击跳空"三阶分类** — 在 0329 第二部分讲到的跳空缺口分类法(攻击型 / 中继型 / 衰竭型),目前 wiki 未明确收录此分类,值得补到 [[关键K]] 或新建一个 [[十张死亡K线图|跳空缺口三阶论]]
- **核心观点**:
  - 主力控盘下不要预测大盘点位,而是看主仓出货的盘面特征
  - 地缘政治推演主要是 Z 哥的兴趣输出,不必当作交易信号
