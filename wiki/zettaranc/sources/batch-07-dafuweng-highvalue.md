大富翁小菜鸟号高价值文件第二批处理（batch-07），聚焦 4 篇体系性长文，涵盖 B2 战法验证、目标价计算、填坑出坑底层逻辑、四不原则年度复盘。

## 核心提炼

### 1. 四分之三阴量 & B2 战法（上篇）

**来源**: `[202508240730]全网最详细Z哥四分之三阴量&B2战法上`

核心贡献：**四分之三阴量假突破识别法** — 突破颈线位次日阴量约为阳量的 3/4 → 大概率假突破（回溯验证 90%+ 成功率）。

- 底层逻辑：真突破 → 主力控盘 → 洗盘必缩量；假突破 → 主力出货 → 阴量不缩
- 牛市真突破概率约 50%，熊市/震荡市仅约 10%
- 止损位设在次日阴线最低点

**编入**: [[B2战法]] 新增「四分之三阴量假突破识别法」章节

### 2. 目标价咋算 — 祖宗之法

**来源**: `[202510110709]温故知新目标价咋算就靠这祖宗之法`

核心贡献：目标价计算实战案例 + 大小坑打法区分。

- 公式：第一目标价 = 坑沿 - 坑底 + 坑沿；第二目标价 = 第一目标价 + 坑深
- 实战案例：米儿(36→63.72)、新前锋(26→37)、李宁(30→57.21)
- 小坑用少妇战法快进快出，大坑靠信仰+基本面拿住不动
- 出坑后回踩坑沿 = 补票机会

**编入**: [[黄金坑三大分类]] 新增「目标价计算实战」「企稳信号」「分类打法总结」章节

### 3. 填坑出坑那点事儿

**来源**: `[202511012358]温故知新聊聊填坑出坑那点事儿`

核心贡献：大坑/小坑/小洞三分类详解 + 三大禁区 + 三类机会。

- 大坑：赚大钱靠信仰，分批建仓(20%→20%→20%=60%)，拿3-5年
- 小坑：赚小钱靠纪律，少妇战法(10%-20%仓位)，1-2月见收益
- 小洞：赚碎银子靠盘感，快进快出一根K线，新手别碰
- 三大禁区：没填过坑的、不懂基本面的、不在主线的
- 三类机会：填完中型坑的、N型完美的、主线出坑的
- 企稳信号：横盘企稳 vs 爬山坡企稳

**编入**: [[黄金坑三大分类]] 融合为「填坑出坑」完整体系

### 4. Z哥2024年大富翁知识点总结

**来源**: `[202509291730]温故知新Z哥2024年大富翁知识点总结梳理`

核心贡献：四不原则详解 + A股四阶段模型 + 只输一根K线 + 核心资产配置。

- **四不原则详解**：不追(924行情教训)、不动(两种情况)、不慌(噪音过滤)、不摸(骚票概念)
- **A股四阶段**：券商点火→公募接棒→老钱进场→散户接盘
- **只输一根K线**：买入K线最低点向下3-5个价位设止损，跌破即走
- **核心资产配置**：罗德里(核心)/新前锋(短期)/斥候(试错)三种角色
- **存款搬家明牌**：全市场看懂的放量长阳 / 安全的底部区域
- **三大优势**：策略优势+成本优势+心理优势

### 5. 异动选股核心逻辑

**来源**: `[202509060645]异动A股选股的核心逻辑与实战指南`

核心贡献：建仓波 vs 一波流区分 + 连续异动信号 + 回踩60日线低吸 + 三大禁区。

- 建仓波特征：连续温和放量，一波比一波强；一波流：单根巨量后无持续
- 连续两三天异动才是真信号，单根放量可能是游资一日游
- 60日线是建仓波"生命线"，回踩不破 = 安全买点
- 三大禁区：高位异动、不止损、用于期货

### 6. 分歧与一致

**来源**: `[202509140801]分歧与一致搞懂这节奏别在行情里瞎冲`

核心贡献：全新概念 — 趋势的"心跳"，分歧→一致循环。

- 上涨趋势：分歧=洗盘（不怕），一致=续涨（别贪）
- 下跌趋势：分歧=反弹（别急），一致=续跌（别碰）
- 板块一致速度：科技快/消费慢/煤炭最慢
- 主力心理战：没出货用分歧磨你，要出货用一致套你
- 买点：分歧后的一致信号（缩量小K线+回踩趋势线）

### 7. 慢牛趋势下的稳健投资体系

**来源**: `[202509160710]浓缩总结慢牛趋势下的稳健投资体系`

核心贡献：慢牛"折线爬楼"模型 + 指数管理工具 + 板块轮动规律。

- 慢牛斜率：每月100-150点，超200点需减仓
- 指数管理工具：低波动/低估值板块承担稳定器功能
- 四季度轮动：概念内轮动→大行业轮动 + "小切大"趋势
- 板块优先级：高成长科技(进攻)→消费顺周期(防守)→大金融(助推)→低波动(对冲)→周期(弹性)
