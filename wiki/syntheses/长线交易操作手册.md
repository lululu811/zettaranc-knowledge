## 研究问题
中长线投资者（持仓周期1-3个月甚至完整牛熊周期）如何运用Zettaranc交易体系进行月度/季度级别的资产配置与波段管理？

## 核心结论
1. 长线交易的核心是**"底仓守信仰，动态仓守纪律"**，两者性质不同、管理规则截然不同
2. **穿越牛熊的关键是防守而非进攻**：先求不亏钱，再求赚大钱
3. **周线格局定方向，日线找买卖点**：长周期有序，短周期服从长周期
4. **牛市拿住、熊市空仓、震荡市波段**：周期判断比个股选择重要10倍
5. **长线不是死扛**：底仓也有止损线（基本面恶化或趋势彻底反转）

## 详细论证

### 一、周期判断：先定牛熊，再谈操作

**第一步：判断当前市场周期**
- **[[周期与人性]]**：A股遵循明显的周期性规律，牛熊交替不可违逆
- **[[活跃市值]]**：判断多空区间的核心指标，确定当前处于牛市、熊市还是震荡市
- **[[白线黄线系统]]**：周线级别的白黄线位置决定大方向
  - 白线在黄线上方且距离拉大 → 牛市确立，重仓持有
  - 白线在黄线下方且距离拉大 → 熊市确立，空仓或极低仓位
  - 白黄纠缠 → 震荡市，波段操作，不赌方向

**不同周期的核心策略**：
| 市场周期 | 仓位建议 | 核心操作 |
|---------|---------|---------|
| 牛市初期 | 5-7成 | 建立底仓，动态仓滚动 |
| 牛市中期 | 7-9成 | 底仓不动，动态仓做T |
| 牛市末期 | 3-5成 | 逐步减仓，准备逃顶 |
| 熊市 | 0-2成 | 空仓为主，偶尔反弹 |
| 震荡市 | 3-5成 | 波段操作，高抛低吸 |

### 二、资产配置：底仓+动态仓的4231阵型

**第二步：建立底仓**
- **底仓定位**：看好行业/公司的长期价值，准备拿完整个牛市周期
- **选股标准**：
  - 只选**[[绝对主线]]**中的核心龙头（如队长、宁子等）
  - 基本面扎实，有长期成长逻辑
  - 周线N型结构完美，大趋势向上
- **底仓纪律**：
  - 除非基本面根本性变化，否则不管短期涨跌都不动
  - 赚的是整个牛市的贝塔收益
  - 底仓不与股票谈恋爱，但与趋势谈恋爱

**第三步：管理动态仓**
- **动态仓定位**：用来做波段、做短线，赚市场波动的钱
- **操作规则**：
  - 严格遵守交易纪律，走坏了立刻拍掉
  - 用**[[双线战法]]**和**[[知行趋势线]]**判断波段高低点
  - 黄线是最终止损线，跌破且收不回必须离场
- **仓位比例**：底仓:动态仓建议为 6:4 或 7:3

### 三、买入：少妇战法的长线化应用

**第四步：择时与选股**
- **择时**：不见兔子不撒鹰，大盘不进入多头区间不开仓
- **选股**：
  - **[[三最原则]]**：只选最美（周线完美图形）、只买最强（主线龙头）、只拿最硬（基本面最扎实）
  - **[[顺周期轮动]]**：按金融→资源→制造→消费的板块轮动规律配置底仓

**第五步：等B1（长线标准）**
- 长线B1要求比短线更严格：
  - **周线B1优先**：日线B1容易受短期波动干扰，周线B1信号更稳
  - **J值标准**：长线底仓可在J值 < 13 时分批建仓，不必等到-10以下（少妇战法严格标准更适合动态仓）
  - **回踩关键支撑位**：年线、半年线、前期平台等重要支撑
- **建仓节奏**：
  - 首次建仓3成（试仓）
  - 确认趋势后加仓至目标仓位
  - 绝不一次性满仓

### 四、持仓：跨越周期的持仓心法

**第六步：底仓持仓纪律**
- **牛市中**：底仓打死不动，任凭洗盘折腾
  - 像钉子一样钉在庄家的粮仓里，成为"硕鼠"
  - 主力灭不掉你，只能带着你一起飞
- **震荡市中**：底仓不动，动态仓高抛低吸
- **熊市中**：底仓也要减仓，甚至清仓（底仓不是死扛的借口）

**第七步：动态仓波段操作**
- **白线当牵牛绳**：动态仓不跌破白线不走
- **半仓放飞**：遇到不确定信号，卖一半动态仓锁定利润
- **去弱留强**：动态仓只留最强的，弱的果断止损换强的

**心态管理**：
- **装睡的拔河**：交易像拔河，绝大部分时间是垃圾时间，等待力量衰竭的临界点
- **建立周线格局**：日线看不懂的切换到周线，周线看不懂的放弃
- **均线只看趋势**：不要把均线当支撑位去精准预测点位，那是极其危险的

### 五、卖出：逃顶与止损

**第八步：长线卖出触发条件**
1. **基本面恶化**：行业逻辑被破坏、公司业绩暴雷 → 底仓也必须清仓
2. **大趋势反转**：周线白线死叉黄线，且持续向下 → 底仓减仓
3. **牛市末期逃顶**：
   - **[[牛市策略]]**中的逃顶信号
   - 市场情绪极度狂热、身边从不炒股的人开始推荐股票
   - 逐步减仓，不要试图卖在最高点
4. **动态仓常规卖出**：
   - 连续两日收盘跌破BBI线清仓
   - S1信号或DSZ战法触发

**卖出纪律**：
- 底仓卖出后不再回头（除非重新出现周线B1）
- 不要试图做"倒T"（卖出后更低买回），长线交易者没有这种能力
- 止盈后即使继续涨也不后悔，只赚认知范围内的钱

### 六、月度/季度复盘

**长线复盘框架**：
- [ ] 当前市场周期判断是否正确？
- [ ] 底仓的基本面是否发生变化？
- [ ] 底仓的周线趋势是否仍然向上？
- [ ] 动态仓的波段操作是否遵循纪律？
- [ ] 仓位配置是否符合当前周期？
- [ ] 是否被短期波动影响了长期判断？

**关键反思问题**：
- 这次交易的周期判断是对是错？
- 底仓和动态仓的界限是否清晰？
- 有没有把底仓当动态仓乱动，或把动态仓当底仓死扛？

## 长线vs短线的核心差异

| 维度 | 短线（1-5天） | 长线（1-3个月+） |
|-----|-------------|----------------|
| 核心工具 | 砖形图、DSZ、量比 | 白线黄线、周线格局 |
| B1标准 | J值 < 13，日线为主 | 周线B1优先，分批建仓 |
| 止损策略 | 单根K线止损（2-3%） | 基本面恶化或趋势反转才止损 |
| 仓位管理 | 动态仓为主，快进快出 | 底仓+动态仓，底仓守信仰 |
| 核心原则 | 择时大于选股、去弱留强 | 防守哲学、框架式交易 |
| 盯盘频率 | 每日盯盘，盘中操作 | 每周复盘，月度调仓 |
| 盈亏比目标 | 10:1 | 3:1 或更高 |
| 资金量建议 | 100万以下 | 100万以上或有稳定现金流 |
| 适用市场环境 | 震荡市、结构性牛市 | 牛市主升浪、完整牛熊周期 |
| 单次仓位 | 单票不超过10% | 底仓单票可达20-30% |
| 持仓数量 | 5-10只（开超市） | 3-5只核心底仓+动态仓 |
| 信息来源 | 分时图、量比、盘口 | 周线图、基本面、行业研报 |
| 情绪管理 | 克服手痒、拒绝噪音 | 装睡的拔河、建立周线格局 |
| 卖出标准 | S1/DSZ/跌破BBI | 基本面恶化/周线趋势反转 |
| 交易频率 | 一周最多两枪 | 一月最多调仓一次 |
