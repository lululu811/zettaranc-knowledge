- **择时**：[[活跃市值]] 判断多空区间，择时永远大于选股
- **选股**：[[三最原则]]（只选最美）、[[顺周期轮动]] 板块节奏
- **买点**：[[B1建仓波]]、[[B2突破]]、[[B3买点]] 三件套
- **卖出**：[[S1信号]] 预警、[[DSZ战法]] 卖出三铁律
- **持仓**：[[去弱留强]]、[[底仓与动态仓]]、[[半仓放飞策略]]
- **风控**：[[防守哲学]]、[[交易闭环]]、[[不可能三角]]

## 使用方式

### Obsidian

1. 在 Obsidian 中点击 **Open folder as vault** → 选择本目录
2. 设置 → 文件与链接 → 附件默认存放路径设为 `assets/`
3. 使用 **关系图谱** 视图查看知识网络连接

详见 [obsidian-config.md](obsidian-config.md)

### 推荐插件

- **Web Clipper** — 网页剪藏
- **Dataview** — 基于 YAML frontmatter 的查询
- **Canvas** — 思维导图
- **Excalidraw** — 手绘风格图表
